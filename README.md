# 🌦 Weather Dashboard

## 📌 Overview
The Weather Dashboard is a responsive web application built using HTML, CSS, and JavaScript. It retrieves real-time weather information from the OpenWeatherMap API using Fetch API and async/await. Users can search for any city and view its current weather details.

## 🚀 Features
- Search weather by city name
- Real-time weather data
- Displays temperature
- Displays humidity
- Displays wind speed
- Displays weather condition
- Fetch API integration
- Async/Await implementation
- JSON parsing
- Error handling for invalid city names and network issues
- Responsive user interface

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await
- REST API
- JSON
- OpenWeatherMap API

## ▶️ How to Run
1. Download or clone the repository.
2. Get a free API key from OpenWeatherMap.
3. Replace `YOUR_API_KEY` in `script.js` with your API key.
4. Open `index.html` in your web browser.
5. Enter a city name and click **Search** to view the current weather.
